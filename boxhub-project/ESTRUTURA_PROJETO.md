## 🚀 Como Usar o Projeto

### **Para Executar:**
1. Abra qualquer arquivo HTML diretamente no navegador:
   - `index.html` - Página principal/Feed
   - `discussions.html` - Discussões
   - `rankings.html` - Rankings
   - `fighters.html` - Lutadores
   - `events.html` - Eventos

### **Para Desenvolver:**
1. Abra a pasta `boxhub-project/` no VS Code
2. Edite os arquivos CSS em `css/`
3. Modifique JavaScript em `js/`
4. Adicione imagens em `assets/images/`
5. Abra os arquivos HTML no navegador para testar

### **Para Testar:**
1. Clique duplo nos arquivos `.html` para abrir no navegador
2. Use as ferramentas de desenvolvedor (F12) para debug
3. Todas as funcionalidades funcionam localmente

### **Para Expandir:**
1. **Adicionar novas funcionalidades**: Edite `js/main.js`
2. **Novos estilos**: Adicione em `css/style.css`
3. **Novas animações**: Use `css/animations.css`
4. **Configurações**: Modifique `js/config.js`
