# BOXHUB - Comunidade de Boxe

Uma plataforma interativa para fãs de boxe compartilharem opiniões, discutirem lutas e criarem rankings da comunidade.

## 📂 Estrutura do Projeto

```
boxhub-project/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos CSS principais
├── js/
│   └── main.js         # JavaScript principal
├── assets/
│   └── images/         # Imagens do projeto
└── README.md           # Este arquivo
```

## 🚀 Funcionalidades

### ✅ Implementadas
- **Sistema de Abas**: Feed, Discussões, Rankings, Lutadores, Eventos
- **Autenticação**: Login e registro de usuários
- **Criação de Posts**: Sistema completo de posts categorizados
- **Sistema de Rankings**: Votação em lutadores com controle único
- **Interações Sociais**: Curtir, comentar e compartilhar posts
- **Interface Responsiva**: Adaptável a diferentes tamanhos de tela
- **Notificações**: Sistema de feedback visual
- **Persistência**: Dados salvos no localStorage do navegador

### 🔧 Em Desenvolvimento
- Sistema de comentários completo
- Upload de imagens
- Sistema de mensagens privadas
- Integração com APIs de boxe
- Sistema de pontuação avançado

## 🎨 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com Grid, Flexbox e animações
- **JavaScript ES6+**: Funcionalidades interativas
- **LocalStorage**: Persistência de dados no cliente

## 🔧 Como Usar

1. **Abrir o projeto**: Simplesmente abra o arquivo `index.html` em qualquer navegador moderno
2. **Navegar**: Use as abas superiores para explorar diferentes seções
3. **Fazer Login**: Clique em "Entrar" ou "Registrar" para acessar funcionalidades completas
4. **Interagir**: Crie posts, vote em lutadores, curta conteúdo

## 🎯 Funcionalidades Principais
